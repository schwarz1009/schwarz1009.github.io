＝2018-09-23＝

二日連続で、今日も英会話教室に行ってきた。今日は、たとえば農場で作っている野菜のような時に、We grow tomatos, apples, and so on.のように、「その他もろもろ」をand so on.やand things like that.やor anything like that.などの言い回しで言うのを習った。今日はレベル7だったせいか、難しかった。

僕は、集中力が高すぎるのが問題だ。それも、常に24時間いつも集中している。そのせいで、何でも会話になる。集中を切ってしまえば、すぐにこの病気は治る。おかしな「集中病」をやっている。

昔は、環境を客観視するのが好きだった。環境を外側から客観視した場合と、環境の一員として環境の中で参加する場合を比べて、その環境に参加するとどうなるか、みんなはその環境を何だと思っているか、ということを分かっていた。環境への適応を知ることで、この世界全ての「哲学的考え方」を分かっていた。そこには統制と反抗があり、全てをボトムアップで変えるための経験的な能力と理性があった。

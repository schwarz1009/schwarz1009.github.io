snap

snapcraftいい感じ

最近、新しく導入したDebian 10で、必要があってsnapアプリケーションを入れたのだが、これが良い感じである。

snapcraftを使うためには、まずaptでsnapdを入れる。

# apt install snapd

そして、snapコマンドでcoreを導入する。

# snap install core

あとは、snap installでパッケージ名を指定して導入すればよい。

# snap install パッケージ名

以下が参考になる。

  • Installing snap on Debian | Snapcraft documentation
  • Installing snapd | Snapcraft documentation

snapアプリケーションの起動と一覧

snapアプリケーションは、/snap/bin/に本来PATHにあるべきファイルがあるので（実際はシンボリックリンク）、以下のように実行する。

$ /snap/bin/appname

頻繁に使うのであれば、PATHに/snap/binを加えよう。

現在導入されているsnapアプリケーションの一覧はsnap listで表示できる。

$ snap list

後日注記：GUIでアプリケーションを起動させることもできる。なので、毎回このようなコマンド入力をする必要は必ずしもない。

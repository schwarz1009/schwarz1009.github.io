Linuxの世界観に戻る

UNIXコマンド（応用２．システム）

UNIXやLinuxのコマンドに関する世界観（応用２．システム）です。UNIXコマンド（応用）やUNIXコマンド（基本）も参照のこと。

システム管理

su -

rootユーザーになってシェルを実行する。

$ su -
Password: (root のパスワードを入力)
# (rootのシェルになった)

終了する時はexitを実行する。

# exit
$ (ユーザーに戻る)

sudo

そのコマンドだけをroot権限で実行出来る。僕はあまり好きではないが、使いやすさを考えてUbuntuでは標準である。

chroot

ルートディレクトリを変更してシェルを実行する。

特に、ディストリビューションの手動によるインストールなどの際、マウントしたディレクトリに入る時などに使われる。

Gentoo Linuxではハンドブックによるインストールの時の必須ツールである。

ファイルシステムデバイス

df

ディスクの使用容量を確認する。Linux アーカイブ・同期・デバイス処理を参照のこと。

du

ファイルやディレクトリの使用容量を確認する。Linux アーカイブ・同期・デバイス処理を参照のこと。

edquota

ユーザのディスク利用容量を制限する。

hdpram/sdpram

ハードディスク・SCSIデバイスの情報を表示・設定する。

partx

パーティション情報を表示する。

ネットワーク操作

lftp

FTPクライアント。コマンドでFTP操作が可能。

詳細はlftpの世界観にあります。

wget

コマンドからURLを直接渡してファイルをダウンロードするためには、wgetを使うのが良い。

links/lynx/w3m

CUI環境でインターネットのWebページを見たい時には、linksやlynxやw3mのようなテキストブラウザが使える。

特に、Gentoo Linuxのインストール時などにハンドブックを確認するのに有用だ。だが、英語でしか文字が表示出来ないCUI環境もあるので注意が必要だ。

curl

コマンドからHTTPリクエストができる。

telnet/ssh

リモートマシンにログインして、リモートでコマンド操作を行う。

ip/ifconfig/iwconfig

ネットワークインターフェースを設定する。低レベルなインターフェース。

ハードウェア情報

lspci

PCIデバイス情報を確認するコマンド。

lsusb

USBデバイスを表示する。

lsscsi

SCSIデバイスを表示する。

lscpu

CPU情報を表示する。

lsblk

ブロックデバイス一覧を表示する。

/proc/cpuinfo, /proc/meminfo

cat /proc/cpuinfoでCPUの情報、cat /proc/meminfoでメモリの情報を確認できる。

busybox

ひとつのバイナリだけで小さく作られた、最低限の小さなUNIXコマンドを実行する。組み込みシステムなどで用いられる。

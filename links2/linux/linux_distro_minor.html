Linuxの世界観に戻る

マイナーな Linux ディストリビューション

マイナーなLinuxのディストリビューションに関する世界観です。

Distrowatch上位

以下は最近のDistrowatch上位ランキング組。

MX Linux

antiXとMEPIS LinuxをベースにしたDebian派生のデスクトップ向けディストリビューション。

MX Linuxも参照のこと。

elementary OS

デスクトップに独自のPantheonを採用した、デザイン性の高いMac風ディストリビューション。かっこいい。

elementary OSも参照のこと。

Voyager Linux

「もうひとつのMac風ディストリビューション」としてelementary OSとともに評価されるセンスの高いディストリビューション。

Solus

デスクトップ環境にBudgieを採用したデスクトップ向けの独自のディストリビューション。超かっこいい。

Budgieも参照のこと。

Zorin OS

Windowsライクなかっこいいディストリビューション。かっこいい。

deepin

デスクトップ環境にDeepinを採用した中国のディストリビューション。

SparkyLinux

軽量なデスクトップ向けディストリビューション。さまざまなデスクトップ環境を簡単に試すことができる。

Bodhi Linux

enlightenmentからforkしたMokshaと呼ばれるデスクトップ環境を使うディストリビューション。

Bodhiとは「菩薩」、Mokshaは「解脱」の意味。enlightenmentはbodhiの英訳であり、「悟り」という意味に近い。

enlightenmentも参照のこと。

SolydXK

DebianベースのXfceとKDE搭載のディストリビューション。

Korora

Fedoraベースのディストリビューションだったが、休止した模様。

KDE neon

KDEプロジェクトがリリースする、最新KDEをいち早く試すことのできるディストリビューション。

KDEも参照のこと。

KaOS

最新KDEのディストリビューション。

Q4OS

デスクトップにTrinity（昔のKDE 3のfork）を採用したディストリビューション。

その他

以下はその他の有名ディストリビューション。

Vector Linux

軽量なディストリビューション。Slackwareベースだが、slapt-getコマンドやGslapt（GUIツール）を使って初心者に優しいパッケージ管理を提供する。

Kali Linux

ペネトレーションテスト（ネットワークで接続されたコンピュータに対し、侵入を試みることで安全性を確認する）に特化したディストリビューション。

Parrot

Debianベースのセキュリティ系ディストリビューション。

Qubes OS

Fedoraベースのセキュリティ系ディストリビューション。

Qubes OSも参照のこと。

Raspberry Pi

Raspberry Piを参照のこと。

Linux From Scratch

正確にはディストリビューションではなく、Linuxをスクラッチから構築する「ディストリビューション作成のレシピ」のようなもの。

Gentooをやると良く分かるが、Linuxディストリビューションは自分で作ろうと思えば簡単に作れる。

その作り方を、ダウンロードとコンパイルの方法としてまとめた文書がLFSであり、その上でさまざまなアプリケーションインストール方法をまとめた文書がBLFS。

LFSは日本語にも訳されているので簡単に試そうと思えば試せる。

僕もやったことがあるが、むしろGentooの方が面白かった。自動化と手動のバランスの極みのようなところがGentooにはあるのだろう。

LFSも参照のこと。

ストリーミング・動画配信など

GStreamer

GStreamerはLinuxのマルチメディアフレームワークで、ストリーミングや動画編集・再生などの基本機能を提供する。

  • GStreamer - Wikipedia

YouTubeで動画配信

言うまでも無いかもしれませんが、自分でストリーミングサーバーを立てるのは現実的ではありません。YouTubeで配信することがおすすめです。

YouTubeを参照のこと。

C++ RTMP Server（crtmpserver）

Linuxにおけるストリーミングサーバーの１つ。自分でYouTubeのような動画配信サイトを作ることができる。

  • Linuxとフリーソフトでライブ動画配信サーバーを構築する - とある技術者の遊戯日記
  • Ubuntuでcrtmpserverを試す（その2） - pochy9n’s diary
  • ストリーミングサーバー C++ RTMP - 実践Linux

nginx + nginx-rtmp-module

nginxにRTMPモジュールを導入することでもストリーミングサーバーを配信することができます。

  • nginxでストリーミングサーバを立ててライブ配信する。 - Qiita
  • 動画配信サーバー構築（nginx+nginx-rtmp-module） - CentOSで自宅サーバー構築

Nginxも参照のこと。

テレビ視聴・録画

MythTV

テレビ放送の視聴や録画が出来る。簡単なレコーダーやそのサーバーを作ることが可能。

  • MythTV - Wikipedia

インターネットラジオ

Gradio/TuneIn

Gradioはインターネットラジオアプリ。TuneIn RadioというWebから世界中のラジオを聴くサービスもあり、自分は主にTuneInでアメリカやヨーロッパのラジオを聴いている。

  • Gradio - Linux App Data
  • TuneIn | 無料インター ネットラジオ | NFL、スポーツ、ポッド キャスト、音楽とニュース

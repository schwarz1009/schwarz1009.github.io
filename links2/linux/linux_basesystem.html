Arch Linuxよりbaseパッケージ

Arch Linuxのbaseパッケージの情報から、baseに含まれるパッケージの各種情報や依存関係を見ることができます。

特に、「View the file list for ...」のリンクをクリックすることで、どのパッケージがどんなファイルを含むのかを知ることができて便利です。

ONLYOFFICE

クラウドで使える共同編集Office

ONLYOFFICEは、オープンソースな新しいOfficeソフトウェア。

MS-Office互換でありながら、クラウド上で共同編集をしたりすることが可能となっている。

日本語の情報は少なめだが、海外では流行っている模様。

  • Install ONLYOFFICE Docs on Fedora Linux with Podman and connect it with Nextcloud - Fedora Magazine
  • ONLYOFFICEを試してみた - Qiita

Docker/PodmanとNextcloud

ONLYOFFICEはDocker/Podmanなどによって導入できる。

また、クラウドサーバであるNextcloudなどに接続して使う。NextcloudはOSSのオンラインストレージ作成ソフトウェア。代わりにownCloudなどを使うこともできる。

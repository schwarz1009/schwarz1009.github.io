bundlerとGemfile

bundlerは、gemの依存関係を解決し、インストールとバージョン管理をしてくれるツール。

bundle installを実行すると、Gemfileに記載されたgemを依存関係も含めてすべてインストールできる。

gemとBundlerの使い方

まず、インストールされているgemを確認するには以下のコマンドを実行する。

gem list

gemをインストールするには以下のようにする。

gem install パッケージ名

gemをアップデートするには以下のようにする。

gem update

Bundlerをインストールするには以下のようにする。

gem install bundler

新しくGemfileを作成するには以下のようにする。

bundle init

Gemfileを以下のように修正する。

source "https://rubygems.org"

gem 'rails'

最後に、以下を実行すればGemfileに記述されたgemがインストールされる。

bundle install

クラスライブラリ・API

VisualBasic2010逆引き大全555の極意を参考に執筆・引用しました。

文字列

文字列の処理はStringクラスから行える。

メソッド説明
IndexOf検索
Contains存在のチェック
Replace置換
Split分割
Remove削除
Insert挿入
Join連結

などのメソッドから操作出来る。

ファイル

ファイル処理はIO.Fileクラスの各メソッドから行える。ディレクトリの処理はIO.Directoryクラス。

File/Directoryクラスは、

メソッド説明
Exists存在確認
Delete削除
Move移動
Copyコピー
RenameFile名前変更

などの操作が出来る。

また、Directoryは

メソッド説明
CreateDirectoryフォルダーの作成
GetDirectoriesフォルダー内の全てのフォルダーを取得
GetFilesフォルダー内の全てのファイルを取得

などの操作が出来る。

テキストファイル

テキストファイルは、IO.File.OpenRead, IO.File.OpenText, IO.StreamReaderなどから読み込みが行なえる。

一行ずつ読み込むためにはReadLineメソッド、一気に全部読み込むためにはReadToEndメソッドを使う。

また、IO.File.OpenWrite, IO.File.AppnedText, IO.File.StreamWriterなどから書き込みが行える。

コモンダイアログ

また、コモンダイアログは、ファイルを開くダイアログがOpenFileDialog、名前をつけて保存するダイアログがSaveFileDialogコンポーネントから開くことができる。

フォントの指定はFontDialog、色の指定はColorDialogから行える。

ネットワークソケット

まず、ソースコードの先頭に

Imports System.Net.Sockets

を追加する。

ネットワーク上のコンピュータにTCP/IPで接続するには、

Dim client As New TcpClient

でクライアントを作って、

client.Connect("localhost", 80)

などとする。

その後に、

Dim stream As NetworkStream = client.GetStream()

でストリームを取得できる。

この状態で、

stream.Write(buffer, 0, buffer.Length)

した内容が、

stream.Read(data, 0, data.Length)

で返ってくる。bufferに送りたいデータを指定し、dataに送られた内容が入る。

そして、

System.Text.Encoding.ASCII.GetString(data)

でテキストを得られる。

サーバーを作りたい場合は、

server = New TcpListener(IPAddress.Loopback, 9000)

として、

server.Start()

とする。Stopで停止する。

この時Threding.Thredクラスを利用してスレッドを作成すること。

実際にクライアントからの接続待ちをする時は、AcceptTcpClientメソッドを呼び出した時で、サーバーがクライアントから接続を受けると、AcceptTcpClientメソッドから戻りTcpClientオブジェクトを取得できる。その上でストリームを取得してReadなどを行なえば良い。

Dim client As TcpClient = server.AcceptTcpClient()
Dim stream As NetworkStream = client.GetStream()

ちなみに、Webサーバーに接続したい時は、ソースコードの先頭に

Imports System.Net

を追加し、WebClientクラスを使用する。OpenRead, DownloadFile, UploadFileなどのメソッドの他、クエリ文字列（URLの?の後に続く文字列）を操作するメソッドなどがある。

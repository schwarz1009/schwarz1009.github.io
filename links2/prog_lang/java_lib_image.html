Javaの世界観３F（画像処理）

Javaによるプログラミングに関する世界観３F（画像処理）です。Javaの世界観３（クラスライブラリ）やJavaの世界観１（言語）やJavaの世界観２（オブジェクト指向）も参照のこと。

Javaで画像処理

ImageIO

Javaで画像を読み書きするためには、ImageIOクラスを使う。

BufferedImage bufferedImage = ImageIO.read(new File(ファイル名));

読み込まれた画像を表示するには、Swingを使う。

public void paint(Graphics graphics) {
  graphics.drawImage(bufferedImage, 0, 0, null);
}

読み込んだ画像データに線を引いたり、文字を書き込むには、BufferedImageオブジェクトからGraphics2Dオブジェクトを作り、Graphics2Dのオブジェクトに線を引いたり文字を書き込む。

  public void paint(Graphics graphics) {
        Graphics2D graphics2D = bufferedImage.createGraphics();
        graphics2D.drawRect(310, 310, 100, 20);
        graphics2D.drawString("RED ROSE", 320, 325);
        graphics.drawImage(bufferedImage, 0, 0, null);
    }

そして、画像を出力するには以下のようにする。

ImageIO.write(bufferedImage, フォーマット名, new File(ファイル名));

フォーマット名にはJPEG, PNG, BMP, WBMP, GIFなどがある。

ビルドツール

Apache Ant

Makeに代わる新しいビルドツール。

GNU makeを置き換えるJava版のビルドツールで、プラットフォーム移植性が高い。

ソースコードの依存関係をXMLで記述する。

  • Apache Ant - Wikipedia

MakeやApacheやJavaアプリケーションサーバやEclipseも参照のこと。

Apache Maven

Apache MavenはJava用のプロジェクト管理やビルドを行うためのツールで、Apache Antの代替技術。

プラグインによる拡張でさまざまな活用ができる。

  • Apache Maven - Wikipedia

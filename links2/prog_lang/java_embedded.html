組み込み・携帯デバイス

Java ME

マイクロコントローラや携帯電話など、組み込み用途に使えるJava。Javaの「どこでも動く」という観点で考えた時に、Windowsで開発したJavaアプリケーションをそのまま組み込み用途に使うことが出来る。C/C++でやる時はクロスコンパイラと呼ばれる技術を使う。

小さなサブセット

Java MEは、Java SEのモバイルや組み込み向けの小さなサブセットか、あるいは最低限のJava機能を搭載した環境。Java SEのすべての機能は利用できない。

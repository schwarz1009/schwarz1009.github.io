JavaでXMLを扱う

DOM

JavaでXMLを扱うためには、DOM（Document Object Model）およびSAX（Simple API For XML）を使う。

SAXは、XMLを読み込みながら、要素をひとつ読み込むごとにイベントとしてプログラムに知らせてくれる。

DOMは、XMLドキュメントを読み込んで解析し、ドキュメントと呼ばれるノードツリーを作る。

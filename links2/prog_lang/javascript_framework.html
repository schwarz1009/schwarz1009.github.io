新技術が目白押し

JavaScriptフレームワークは、新技術が目白押しです。

たとえばAjaxやjQueryのように、ブラウザで動かすためにもっとデザイナブルかつフレシキブルな効果を与えるものから、サーバーサイドのNode.js、Web技術でデスクトップアプリケーションを開発できるElectron、AltJSとして注目されるTypeScript、React + ReduxやVue.js + Vuexのようなモダンなフレームワークなど、多種多様です。

こうしたJavaScriptの最先端の技術は、今まさに開拓途上であり、数年後どうなっているかは分かりません。ですが、たくさんの応用技術があり、他の（サーバーサイドの）言語と比べても面白い分野ではないかと思います。

npmの使い方

以下はJavaScript フレームワーク入門を参考に執筆しました。

基本的に、

npm install packagename

でパッケージのローカルインストール（クライアントサイドで利用する時に使う）ができる。また、

npm install -g packagename

でグローバルインストール（全てのNode.jsプロジェクトから参照できる。サーバーサイドで利用する時に使う）ができる。また、

npm update -g

で全パッケージの更新ができる。

インストールされるのはJavaScriptファイルなので、スクリプトファイルになっている。基本的に、「node_modules」内に保存されるパッケージの中から、必要なスクリプトファイルを探して、<script>タグで読み込めばよい。

また、npmでは、このようなコマンド一発のインストールではなく、プロジェクトに必要なパッケージの情報ファイルであるpackage.jsonに記述してパッケージをインストールすることもできる。この場合、JSON形式でパッケージの情報を記述する。

プログレッシブWebアプリ（PWA）

PWAとは「モバイルアプリのように使えるWebサイト」で、サイトにアクセスする感覚＋ローカルにインストールする感覚という「オンラインとオフラインを融合させた」感覚で、プッシュ通知も使えてインターネットに接続されていなくてもアプリを使用できるなどのアプリ並みの機能を持ちながら、サイトにアクセスするだけで使用することができる。

PWAでは、Service Workerと呼ばれる機構を使用することで、そうしたアプリ並みの機能をオンラインのWebサイトに持たせることを可能としている。

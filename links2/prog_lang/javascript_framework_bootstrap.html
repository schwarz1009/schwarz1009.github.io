CSSのスタイル付けが面倒なら、とりあえずBootstrapで

実際、HTMLとCSSをゼロから用意するのは面倒くさい作業です。

Bootstrapを使えば、最初からスタイライズされたCSSを簡単に使うことができます。

ボタンやメニューだけではなく、コンテナやスペーシングのような部品がそろっていて、セレクタに設定するだけで即座に使えます。

Tailwind CSS/UnoCSS

Tailwind CSS/UnoCSSはCSSフレームワークあるいはエンジン。Bootstrapのように既にボタンやメニューがデザインされておらず、自分でデザインを行う。

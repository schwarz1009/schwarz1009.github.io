入出力・ファイルAPI

FileReader、FileWriterを使用して文字データの入出力を行う場合、デフォルトのエンコード方式が使われる。

エンコード方式を指定してファイル入出力処理を行いたい場合は、InputStreamReader, FileInputStreamとOutputStreamWriter, FileOutputStreamクラスを組み合わせて行う。

コマンドラインでの入出力

Javaで入出力をする時は、ストリーム系のクラスを使う。入出力はコマンドラインの基本です。頑張って理解しましょう。

  • [java.io] Java文字列入力のまとめ (InputStream, Reader, Scanner) - Qiita

たとえば、入力は、

BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
String line = reader.readLine();

とする。出力はSystem.out.println()で行えばよい。

ファイル操作

Javaでファイル操作をする時は、バッファ系のクラスを利用する。入出力とファイル処理はコマンドラインの基本であるため、要勉強。

ファイルの読み込みはFileReaderを用いるが、BurfferedReaderという「バッファリングを処理するクラス」を一枚被せて利用することが多い。

ちなみに、FileReaderの親クラスはjava.io.InputStreamReader。

メソッドはreadLineとcloseを使う。

  • BufferedReader／BufferedWriterクラス | Javaコード入門

たとえば、

BufferedReader reader = new BufferedReader(new FileReader(filename));
String line = reader.readLine();

とする。

ファイルの書き込みはFileWriterを用いる。BufferedWriterは書き込みの際にバッファリングを行うクラス。PrintWriterは、printlnという便利なメソッドを持つクラス。

  • 便利な書き込み用クラス - テキストファイルの入出力 - Java入門

たとえば、

BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
PrintWriter writer = new PrintWriter(new BufferedWriter(new FileWriter(filename)));
String line;
while ((line = reader.readLine()) != null) {
    writer.println(line);
}

のようにすればよい。

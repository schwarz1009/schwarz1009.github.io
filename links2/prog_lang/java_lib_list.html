イテレータ

Iteratorインタフェースは、コレクション内の要素に順番にアクセスする手段を提供する。

以下のようになる。

for (Iterator i = c.iterator(); i.hasNext();) {
    i.next();
    i.remove();
}

Iteratorインターフェースは、次の要素があるかどうかの確認であるhasNext()、次の要素を返すnext()、最後の要素を削除するremove()を使うことができる。

Listインタフェースを実装したクラスの場合、機能を拡張したListIteratorを利用出来る。Iteratorは後方への順次アクセスのみをサポートしているが、ListIteratorは前方への順次アクセスもできる。また、Iteratorでは削除しかできないが、ListIteratorは追加・変更もできる。ListIteratorは現在位置を知らせるメソッドもある。

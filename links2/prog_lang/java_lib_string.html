Javaによるプログラミングに関する世界観３C（文字列）です。Javaの世界観３（クラスライブラリ）やJavaの世界観１（言語）やJavaの世界観２（オブジェクト指向）も参照のこと。

String（文字列型）

Stringクラス

Javaでの文字列は、以下のように記述する。

String s = "文字列";

この、ダブルクォーテーションで囲まれた文字列を記述した時点で、既に「new String()」相当のStringインスタンスが作られている。以下のような記述もできるが、2つのインスタンスが生まれてしまうため、以下のような記述は非効率的で、避けた方が良い。

String s = new String("文字列");
  • 【Java】 そもそもStringって何？ | 一番かんたんなJava入門

後日注記：Javaは文字列が参照型であるため、==で文字列を比較することはできない。equalsメソッドを使う必要がある。

分割のためのメソッド

JavaのStringでとても便利なメソッドとして、splitメソッドが挙げられる。ある正規表現で区切られた文字列を分割することができる。

  • 文字列を分割する(split) - Stringクラス - Let'sプログラミング

置換のためのメソッド

replaceFirstメソッドは最初に該当した文字列を置換、replaceAllメソッドは該当する全ての文字列を置換する。

str.replaceFirst(regex, repl)
str.replaceAll(regex, repl)

とする。光を闇に変換するには、

str.replaceAll("光", "闇")

とする。

同様の操作は、

Pattern.compile(regex).matcher(str).replaceFirst(repl)

や、

Pattern.compile(regex).matcher(str).replaceAll(repl)

でも実現できる。

  • 最初に該当した文字列を置換する(replaceFirst) - Stringクラス - Let'sプログラミング
  • 該当する全ての文字列を置換する(replaceAll) - Stringクラス - Let'sプログラミング

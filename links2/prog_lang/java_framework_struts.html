Webフレームワーク

Apache Struts

JavaによるWebフレームワークで、高い生産性を持つMVCフレームワークの一つ。

RubyにおけるRailsやPHPにおけるCakePHPやPythonにおけるDjangoと違い、あまり標準になりきれていない印象を受ける。Springのようなフレームワークも広く知られている。

余談

Microsoft SQL Serverと一緒に使おう

ADO.NETは、Microsoft SQL Serverと一緒に使いましょう。

ADO.NETは、.NET Frameworkでデータベース技術を使う際に必要な技術です。

そして、Windowsでのデータベースと言えば、Microsoft SQL Serverです。

Microsoft SQL Serverは、Windowsに特化して開発されており、商用のSQLサーバーとしてはOracleに次ぐシェアがあります。

多くのWindowsを使ったITシステムの開発で、ADO.NETは必須です。特に、ASP.NETと一緒に使われることが多いです。

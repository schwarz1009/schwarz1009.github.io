プログラミング言語の世界観に戻る

C#の世界観

C#に関する世界観です。.NETも参照のこと。

C#

僕はC#が大好き

実際のところ、僕はC#が大好きである。

そもそも、僕はDelphiが好きだった。昔、Delphi 6のフリー版を使ってOpenJaneなどをコンパイルし、その後にDelphi 7の有料版を買った。

C#は、Microsoftが.NET Frameworkを開発した時に作った言語で、Delphiの主要な開発者を引き抜いて設計した。

そのこともあってか、C#や.NET FrameworkにはDelphiの影響が随所にみられる。特にフォームデザイナーの仕様などはそれそのものである。

Javaを参考にした.NETの言語

C#はJavaを参考にした.NETの言語である。

Javaとの違いは、プロパティの実現やデリゲート・無名関数、LINQ、yield returnなどもあるが、大きく違うのは、「WindowsのネイティブGUIに対応」したことと、「.NET Framerowkの上で動くVBやF#などの多様な言語を使える」ということである。

Windows.Forms

C#のWindows.Formsは、とても美しく、数学的である。JavaではSwingというPure JavaのGUIを使うが、ルック＆フィールがとても独自で個性的、悪く言えば醜い。C#やVB.NETを使えば、Windowsのネイティブなアプリケーションを開発できる。

Windows.Formsは大人気で、「Windows APIやMFCに比べて分かりやすく、可読性も高く美しい」と評判である。Windows APIのメッセージループのコードなどがとにかく醜かったWindowsだが、C#によって一転とても美しいAPIに変わった。

.NETではVBやF#も使える

また、.NETではVBやF#、Managed C++を使うこともできる。これらは同じ.NET Frameworkのライブラリを使っていて、一度覚えれば他の言語でも同じように使える。

Windows以外にはmonoがある

Windows以外のプラットフォームとしてはmonoがある。monoでGTK#を使うことで、GTKでもC#でプログラミングを行うことができる。

UnityでもC#を使う

ゲームエンジンのUnityでも、monoを主要プラットフォームとしているためC#を用いてゲームプログラミングを行う。

とても洗練されたGUI開発ができる

実際のところ、C#とVisual Studioと.NETを使うことで、とても洗練されたGUIアプリケーション開発ができます。

Microsoftは賢いです。特に、Visual StudioのフォームデザイナーとC#によるイベントドリブンを体験すると、感動します。本当に「最高のプログラミング環境」になります。

僕が思うに、Delphiの開発メンバーを引き抜いたことが大きく貢献していると思います。C#は最高です。

その他の情報

.NETにC#関係の内容があるので参照してください。

C#のイベント駆動

イベントハンドラとデリゲート

イベントハンドラはデリゲートの1つで、C#におけるイベント駆動で、コールバック関数をイベントに追加するのに使う。

デリゲートとは、関数を格納できる変数のこと。

まず、以下のようなコードがあったとする。

class Form1 : Form
{
    Button button1;
    
    ...
}

ここで、以下のようなコールバック関数がある。

    private void button1_Click(object sender, EventArgs e)
    {
       MessageBox.Show("Hello");
    }

この時、以下のようにEventHandlerを用いてイベントを登録できる。

    public Form1()
    {
        button1 = new Button()
        {
            Text = "クリックしてね",
            Location = new Point(10, 10),
            Size = new Size(160, 40),
        };

        // イベントハンドラを登録
        button1.Click += new EventHandler(button1_Click);

        this.Controls.Add(button1);
    }

        public List<WebBrowser> alWebs;

        public Form1()
        {
            InitializeComponent();
            alWebs = new List<WebBrowser>();
        }

        private void toolStripButton1_Click(object sender, EventArgs e)
        {
            // タブコントロールに新しいタブとブラウザを作成する
            WebBrowser web;
            TabPage tabPageN;

            alWebs.Add(web);
            web.Dock = DockStyle.Fill;
            
            tabPageN = new TabPage();
            tabPageN.Controls.Add(web);
            tabControl1.Controls.Add(tabPageN);

        }
    }

この前自分で作った、<p>にid="行番号"をつけるプログラム。

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.IO;
using System.Text.RegularExpressions;

namespace ConsoleApplication1
{
    class Program
    {
        static void Main(string[] args)
        {
            RepA("st.html");
            RepA("pm.html");
            RepA("tx.html");
            RepA("hs.html");
            RepA("mm.html");
            RepA("sk.html");
            RepA("lt.html");
            RepA("cp.html");
            RepA("sc.html");
            RepA("etc.html");
            RepA("hs2.html");
            RepB("lt2.html");
            RepB("sk2.html");
            RepB("hs3.html");
            RepB("cp2.html");
            RepB("lg.html");
            RepB("sc2.html");
            RepB("tg.html");
            RepB("ms.html");
            RepB("etc2.html");
            RepB("tg2.html");
            RepB("sk3.html");
            RepB("cp3.html");
            RepB("lt3.html");
            RepB("tx2.html");
        }
        static void RepA(string strin)
        {
            string str = "";
            Encoding enc = Encoding.GetEncoding(932);
            using (StreamReader r = new StreamReader(strin, enc))
            {
                string line;
                int c = 0;
                while ((line = r.ReadLine()) != null) // 1行ずつ読み出し。
                {
                    string mc;
                    if (Regex.IsMatch(line, "----"))
                    {
                        mc = Regex.Replace(line, @"", "<p id=\"" + (c + 1) + "\">");
                        str += mc + "\n";
                        c++;
                    }
                    else
                    {
                        mc = line;
                        str += mc + "\n";
                        c++;
                    }
                }
            }
            // ファイルにテキストを書き出し。
            using (StreamWriter w = new StreamWriter(strin + ".o.html", false, enc))
            {
                w.WriteLine(str);
            }
        }

        static void RepB(string strin)
        {
            string str = "";
            Encoding enc = Encoding.GetEncoding(932);
            using (StreamReader r = new StreamReader(strin, enc))
            {
                string line;
                int c = 0;
                while ((line = r.ReadLine()) != null) // 1行ずつ読み出し。
                {
                    string mc;
                    if (Regex.IsMatch(line, "◇◇"))
                    {
                        mc = Regex.Replace(line, @"", "<p id=\"" + (c + 1) + "\">");
                        str += mc + "\n";
                        c++;
                    }
                    else
                    {
                        mc = line;
                        str += mc + "\n";
                        c++;
                    }
                }
            }
            // ファイルにテキストを書き出し。
            using (StreamWriter w = new StreamWriter(strin + ".o.html", false, enc))
            {
                w.WriteLine(str);
            }
        }
    }
}

正規表現のメモ。

検索：<p id="\(.*\)">----\n<p>\(.*\)

検索：<p id="\(.*\)">◇◇\n<p>\(.*\)

置換：<a href="tx2.html#\1">\2</a>,\n

検索：<p>[\n|].*$

検索：<p>.*$

検索：^\n$

検索： ..:..

----か◇◇で始まる行を、その行のpタグのid属性と次の行の内容でリンクに変換します。

React

React

Facebookが中心となって開発されているJavaScriptフレームワーク。Angularと同様、コンポーネント指向だが、よりシンプルで、画面表示（View）に特化して設計されている。

「JSX」という技術を用いることで、JavaScriptからHTML要素を生成し、HTMLと同じようなタグを直接記述できる。また、ソースコードをJavaScriptに変換するBabelと呼ばれるトランスコンパイラ言語を採用している。

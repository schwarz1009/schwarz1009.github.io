AWS (Amazon Web Services/Amazon EC2)

Amazon Web Services (AWS) は、Amazon.comによるクラウドプラットフォーム。

Amazon EC2は、Amazonの巨大データセンターのリソースを使ったクラウドのレンタルサーバー。

Amazon EC2 & AWSは流行っている

Qiitaなどを見る限りには、Amazon EC2とAWSはとても流行しています。

自分で独自のサービスを作って、仕事をするだけではなく調査研究などを行う上でも、EC2上で行うことが増えています。

EC2では、インスタンスという単位でサーバーを構築します。スケールの変更が容易にでき、また値段はサーバーの使用時間によって払うなど複数のプランがあります。

OSについては、現在はRed Hat Enterprise LinuxやSUSE Linux Enterprise Serverなどを有償で、Fedora、Ubuntu、openSUSEなどを無償で利用でき、日本のサーバーは東京にあります。

AWSの主なサービス

（【インスタンスって何??】AWSでよく聞く単語まとめてみた - Qiitaを参考に執筆しました。）

Amazon EC2 (Amazon Elastic Compute Cloud)

従量課金制のレンタル仮想サーバ。

S3 (Simple Storage Service)

ストレージサービス。「イレブンナインの耐久性」で有名で、99.999999999%データが消失することがない。

AMI (Amazon Machine Image)

EC2で使われるディスクイメージ。

ELB (Elastic Load Balancing)

ロードバランサー（負荷分散）のためのサービス。

RDS (Amazon Relational Database Service)

データベースサービス。

簡単に何でもできる仮想サーバーが手に入る

AWSやAmazon EC2の利点は、アカウントを登録して、クレジットカードを登録して、従量課金制の契約を行えば、何でもできる仮想サーバーが手に入る、ということです。

まさに、手軽に登録することができ、簡単にサーバーが手に入ります。その代り、きちんと払うべき対価はクレジットカードを通じて銀行から引き落としされます。

ただ、僕も実際にEC2を買ったことが無いため、実際のところ、どれだけ面倒くさいか簡単かは分かりません。詳しくは以下を参照のこと。

Amazonのクラウド技術

Lambda

AWS Lambdaは、サーバーレスコンピューティングのプラットフォーム。従来のWebアプリケーションのようにサーバを必要とせず、実行処理だけを記述して利用でき、イベントに応じて実行処理を実行できる。

Kinesis

Kinesisはストリーミングデータの処理機能。

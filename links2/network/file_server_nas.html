NAS

NAS（Network Attached Storage）は、ネットワーク（LAN）に直接つながったストレージ（ハードディスク）のこと。

ネットワーク内のどのコンピュータからもストレージにアクセスでき、家庭内・企業内のファイル共有に使うことができる。

基本的にファイルサーバーと同じだが、NASはストレージに特化しているため導入が簡単で、パソコンを常時起動するよりもよりも消費電力が少ないため、経済的。

中小の企業内などでファイル共有をしたい場合は、NASを購入して使うのが手軽でおすすめである。

ただし、ファイルを共有するため、誰かが勝手にファイルを削除したりすることも考えられる。NASでは、それぞれのユーザーにアクセス権を設定するなどの機能がある。

NASのOSにはLinuxやWindows Storage Serverなどが使われることが多く、NAS用のOSとして、FreeBSDをベースとしたFreeNASなどのOSが知られている。

RAID

RAIDはNASに備わっていることが多いストレージの機能で、複数のハードディスクをひとつの仮想ハードディスクとして扱い、ひとつのストレージが壊れても別のストレージにデータが残るように「冗長化」することのできる機構。

大企業などでは、大容量・多ユーザーでファイル共有をするため、RAIDが備わっているとありがたい。だが、あくまで冗長化の仕組みのひとつであり、RAIDがあれば絶対に安心してバックアップがとれるわけではない。

後日注記：ファイルサーバを構築するにあたって、「もしサーバが壊れたらどうしよう」と心配するのは、多くの場合必要ない。多くのサーバでは、RAIDによって冗長化構成がなされているため、ほとんどの場合データは壊れない。本当に100%に近い数値でインフラが稼働するとか、絶対に破損してはならないサーバでない限り、データが壊れることはない。

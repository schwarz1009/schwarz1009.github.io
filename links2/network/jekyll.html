Jekyll

Jekyllとは

Jekyllは、静的サイトジェネレータ。ブログのようなリッチなサイトを静的なHTMLページを自動生成することで可能とする。

  • Jekyll - Wikipedia

GitHub Pagesで採用されている

Jekyllは、GitHubの静的ホームページ公開サービスであるGitHub Pagesで採用されている。

Gitを参照のこと。

Rubyで書かれている

JekyllはRubyで書かれている。導入の手順は公式サイトを参照のこと。

ソフトウェア（ビジネス・書類）の世界観

WindowsなどのOS向けのサードパーティ製ソフトウェア（ビジネス・書類）の世界観です。

GIMPやLibreOfficeなど、クロスプラットフォームなもの（UNIXやLinuxでも動くもの）も含まれています。

Webサービスも参照のこと。

ビジネス・書類

Officeスイート

Microsoft Office

Microsoft製Officeスイート。僕はMS-Office世代なので、反Office的なアレルギーはありません。Officeは使えます。

LibreOffice

オープンソースのOfficeスイート。とても良いのだがMS-Officeとは別物だと割り切って使う必要がある。

  • LibreOffice

一太郎

ジャストシステムの有料Officeスイート。

テキストエディタ

Vim

UNIXで標準的なテキストエディタ。

Emacs

Lispで拡張可能なテキストエディタ。

xyzzy

Windowsで使えるEmacsと良く似たテキストエディタ。

xyzzyを参照のこと。

サクラエディタ

Windows向けの高機能なエディタ。

サクラエディタを参照のこと。

秀丸エディタ

商用のテキストエディタ。

Atom

GitHubが開発したオープンソースのテキストエディタ。自分は作業所のMacではAtomを使っている。

もう既にEmacsやviは古い。今はAtomである。

AtomはElectron（GitHubが開発したオープンソースのソフトウェアフレームワーク）を使っている。

ElectronはChromiumとNode.jsを使うことで、HTML、CSS、JavaScriptのようなWeb技術で、macOS、Windows、Linuxに対応したデスクトップアプリケーションをつくることができる。

  • Atom - Wikipedia
  • Electron (ソフトウェア) - Wikipedia

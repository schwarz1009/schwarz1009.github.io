Windows/PCの世界観に戻る

ソフトウェア（ネットワーク）の世界観

WindowsなどのOS向けのサードパーティ製ソフトウェア（ネットワーク）の世界観です。

ネットワーク

Webブラウザ

Internet Explorer/Microsoft Edge

Microsoft製Webブラウザ。昔はIEエンジンを使ったさまざまなタブブラウザがあった。

Sleipnir

国産のWebブラウザ。昔、タブブラウザを一通りインストールしまくっていた時に使っていた。

Google Chrome

GoogleによるWebブラウザ。昔メインブラウザだった。軽くてGoogleアカウントでブックマークを同期できてよかった。

Mozilla Firefox

MozillaによるWebブラウザ。最近のメインブラウザ。

Opera

OperaによるWebブラウザ。昔適当に使っていた。

FTPクライアント

FFFTP

FTPクライアント。

WinSCP

FTPクライアント。僕はホームページのファイル転送にWinSCPを使っている。

メール

Outlook

Microsoft製メーラー。

Mozilla Thunderbird

Mozillaによるメーラー。

Exchange

Microsoft製グループウェア・サーバー。

Webメール

Yahoo!メール

Yahoo!のメールサービス。

Gmail

Googleのメールサービス。

メッセンジャー・IRC

IRC

チャット。

Windows、Mac、Linuxなどに向けた、さまざまなIRCクライアントがある。

同じチャンネル（小部屋）に居る人々と簡単なチャットが出来る。

Linuxに関する質問をすると、すぐに返ってくることも多い。

だが、LinuxなどのプロジェクトのWebサイトなどに載っているIRCチャンネルは英語であることが多いので、日本人は日本語専用のチャンネルを使いましょう。

サーバー

Apache httpd

オープンソースのWebサーバー。

IIS

WindowsのWebサーバー。Apacheとシェアを競い合っている。
